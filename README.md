# B24 Metric

Модуль для Bitrix24, который собирает и отображает ежедневные метрики сотрудников:
касания, назначенные и проведённые встречи, новые КП и оплаты.

## Что делает модуль

- Хранит метрики по сотрудникам и датам.
- Показывает статистику за выбранный период.
- Поддерживает суммирование показателей по нескольким сотрудникам.
- Строит графики с линией тренда (если в выборке 6+ точек).

## Структура проекта

- `index.php` — точка входа в Bitrix.
- `Metric.php` — контроллер страницы и обработка действий.
- `config.php` — окружение, подключение ресурсов, базовые настройки.
- `functions.php` — работа с БД и бизнес-логика.
- `view.php` — разметка, интерфейс и клиентский JS.
- `css/metrics.css` — стили.
- `js/Chart.min.js` — библиотека графиков.

## Модель данных (основная таблица)

```sql
b_aktivnosti_metrics
(
  ID                  INT AUTO_INCREMENT PRIMARY KEY,
  USER_ID             INT NOT NULL,
  METRIC_DATE         DATE NOT NULL,
  TOUCHES             INT,
  MEETINGS_SCHEDULED  INT,
  MEETINGS_HELD       INT,
  PROPOSALS_NEW       INT,
  PAYMENTS            INT,
  UNIQUE KEY ux_uid_date (USER_ID, METRIC_DATE)
)
```

## Поведение хранения данных

- Для каждой пары `USER_ID + METRIC_DATE` хранится одна строка.
- При повторном вводе за ту же дату запись обновляется, а не дублируется.
- При инициализации поддерживается консистентность данных и уникальный индекс `ux_uid_date`.

## Важные замечания

- Модуль рассчитан на запуск внутри окружения Bitrix (`B_PROLOG_INCLUDED`).
- Для корректной работы требуются права на чтение/запись в каталог модуля (ресурсы `js/css`).
